Easy to create multiple newsletters containing the blog latest posts.

== Main Features ==
1. Setup multiple newsletters according to the post categories.
2. Different send modes include manual, weekly and monthly.
3. Custom newsletter content, template and post count.
4. Newsletter signup widget for user registration. 
5. Subscriber info list containing email, ip and registeration time. 

**More info:**

* [enl-newsletter](http://www.wp-coder.net/)
* Find out more great [WordPress plugins](http://www.wp-coder.net/)

== Installation ==

1. Upload the contents of enl-newsletter.zip to your plugins directory.
2. Activate the plugin
3. There will be a "Newsletter" menu on your admin panel. 
4. Create a new campaign on the "add campaign" menu.
5. Active the "Enl newsletter" widget on your "Appearance->Widget" menu.
6. That's all. You newsletter will be send to users according to your configuration!
